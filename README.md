# Báo cáo môn Kiến trúc và thiết kế phần mềm

> Ngày 18/04/2026

- Modul Quản lý trang phục
- Modul Nhập trang phục từ nhà cung cấp

Kiến trúc: Web Service

Backend: Spring Boot

Frontend: ReactJS

Database: MySQL

Cách chạy: `docker compose up --build`

Truy cập

- `http://localhost:3000` để xem giao diện Client
- `http://localhost:8081` để xem cơ sở dữ liệu
